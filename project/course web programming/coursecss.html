CSS Tutorials

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan layout halaman web. CSS memungkinkan Anda untuk mengubah warna, font, spasi, dan elemen visual lainnya.

Dasar-dasar CSS

  • Selektor CSS: Digunakan untuk memilih elemen HTML yang akan diberi gaya.
  • Properti CSS: Menentukan aspek visual yang akan diubah, seperti warna, ukuran, dan margin.
  • Nilai CSS: Menentukan nilai untuk properti tertentu, seperti "red" untuk warna atau "16px" untuk ukuran font.
