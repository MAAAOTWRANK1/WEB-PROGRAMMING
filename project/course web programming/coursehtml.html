HTML Tutorials

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML digunakan untuk membuat struktur dan konten halaman web, seperti teks, gambar, tautan, dan elemen lainnya.

Dasar-dasar HTML

  • Elemen HTML: Struktur dasar dari sebuah halaman web.
  • Tag HTML: Digunakan untuk membungkus konten dan memberikan makna.
  • Atribut HTML: Menyediakan informasi tambahan tentang elemen.
